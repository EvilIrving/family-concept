# Release 内容同步约束

英语 `en-US` 是 App Store 上架资料的主语言和事实源。

更新 `en-US/metadata.md` 后，必须同步更新 `zh-Hans/metadata.md`、`ja/metadata.md` 和 `ko/metadata.md`，确保字段结构、含义、发布口径和资源说明一致。

新增、删除或重命名任何上架字段时，必须同时应用到全部语言目录。

资源目录结构在各语言下保持一致：`assets/icons/` 存放图标，`assets/screenshots/` 存放截图。

本目录中的文案以 App Store Connect 可填写内容为准，不放实现细节或开发记录。
