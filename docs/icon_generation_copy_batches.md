# 图形生成批次复制稿

每批仅需复制下方 text 框内全文。**说明：多数扩散生图并不会输出真 PNG Alpha，却会误把「透明」画成灰白棋盘格**当作示意。正文已改为：**显式禁止棋盘格**，并写明若无法生成透明底则改用**单色键控背景（文中 #FF00FF 品红）**以便在 Figma/Photoshop/Affinity 等中一键抠除，再上屏前导出带 Alpha。

下列清单为 `kitchen` 内语义、状态与代码视图的对照表，与下文第 1–26 批生成主题对应（**第 21–26 批**为源代码审计补充分拆：偏好与裁切流程、历史订单 UI、占位与 Tab、点餐条等）。
