## 当前功能

- 邮箱登录
- 邮箱 + 用户名 + 密码注册
- 注册页内联完成加入家庭或创建家庭
- 当前家庭切换
- 家庭菜单双列浏览、搜索、分类筛选、直接加减菜
- 菜品新增 / 编辑 / 归档 / 图片上传
- Orders 页直接展示当前订单
- 当前订单加菜、下单、结束订单、采购清单
- 设置页家庭信息、邀请码操作、个人资料 sheet、成员管理 sheet、历史订单 sheet
