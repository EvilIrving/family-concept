# 项目 AI Agent 使用规范

## 文档同步要求

-  
- `docs/` 是与代码实时对齐的事实来源
- `README.md` 是产品说明，是对用户解释的。
- `CLAUDE.md` and `AGENTS.md` 是AI Agent 使用规范，是给AI Agent 看的，两个文件必须保持一致内容

交互设计：

操作像现实一样直接（Direct Manipulation）

反馈几乎是瞬时的（Immediate Feedback）

出错可以轻松恢复（Forgiveness）

整个过程连续无阻（Continuity）

无阻塞（少弹窗）

无中断（少等待）

无跳跃（连续过渡）

少一些page，可以多使用 sheet方式 填写信息
